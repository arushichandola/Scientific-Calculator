# Scientific-Calculator

The Scientific Calculator is a Java-based console program designed to implement 32 operations 
of mathematics. The program is menu-driven and thus establishes user – friendly interaction. 

The calculator supports the following functionalities: 
1. Addition 
2. Subtraction 
3. Multiplication 
4. Division 
5. Modulus (Absolute value) 
6. Square 
7. Cube 
8. Square root 
9. Cube root 
10. Power 
11. Exponential function 
12. Logarithmic 
13. Natural Logarithmic 
14. Sine 
15. Cosine 
16. Tangent 
17. Sine inverse 
18. Cosine inverse 
19. Tangent inverse 
20. Factorial 
21. PI 
22. e  
23. Degree to Radians 
24. Radians to Degree 
25. Mean 
26. Median 
27. Mode 
28. Permutation 
29. Combination 
30. Prime number check 
31. LCM 
32. HCF
    
The calculator can also handle some exceptional cases, simultaneously providing appropriate 
output statements.
